Connect Google Calendar

✓ The Google Calendar API is completely free — no subscription needed. You only need a free Google account.

One-time setup (about 5 minutes):

  1. Go to console.cloud.google.com and sign in with any Google account. Click Select a project → New Project, give it any name, click Create.
  2. In the search bar type Google Calendar API, click it, then click Enable.
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Choose Web application. Under "Authorised JavaScript origins" add null (for local files) or your hosted URL. Click Create.
  4. Copy the Client ID shown (ends in .apps.googleusercontent.com) and paste it below.
